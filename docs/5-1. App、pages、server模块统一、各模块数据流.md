当前全局状态识别
Auth 状态：用户信息、Token、角色。目前散落在 
useAuth
 hook 中。
业务数据：广告列表。目前主要由 
useAdsData
 管理并在顶层注入。
UI 状态：全局 Toast、全局 Loading。
推荐方案：Zustand + React Query
Zustand：管理 Auth 状态和简单的 UI 全局状态（如 Modal 状态、侧边栏收起）。重量级轻，学习成本低。
React Query：管理服务器状态（Ads 数据）。自带缓存、自动刷新和 Loading/Error 处理，极大简化 
useAdsData
 逻辑。
